About AI Study Pal

Project Overview

AI Study Pal is a simplified web-based application designed for students to apply Artificial Intelligence Course Curriculum topics. It creates a user-friendly study assistant that generates basic study plans, summarizes texts, creates practice quizzes, and provides encouraging feedback.

Tech Stack

  • Language: Python 3.x
  • Web Framework: Flask
  • Data Handling: Pandas
  • Frontend: HTML5, CSS3 (Glassmorphism)

AI Models

  • Quiz Generation: Logistic Regression (Classification), K-Means (Clustering)
  • Summarization: LSTM Neural Network (Seq2Seq) / Heuristic Fallback
  • NLP: NLTK for Tokenization and Keyword Extraction

Ethics & Privacy

This project is designed with a positive educational focus. No personal user data is permanently stored or shared. All models operate locally for demonstration purposes.
